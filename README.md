# victorian-road-crash-analysis
Real-world road crash data analysis using Python, Pandas, Matplotlib and Seaborn to identify transport safety trends across Victoria
## Overview
Analysed a real-world Victorian road crash dataset to surface operational trends 
supporting transport safety decisions. The project involved data cleaning, feature 
engineering, and visualisation to identify patterns in crash frequency and severity 
across Victoria.

## Files
- `Assignment3.ipynb` — Main Python notebook containing all analysis, visualisations and findings
- `S4216466_Assignment3_Presentation.pptx` — Presentation summarising key insights and charts

## What I Did
- Cleaned and preprocessed raw crash data using Pandas — removed duplicates, 
  handled missing values, standardised date/time fields
- Engineered new features (hour of day, day of week) from raw timestamps 
  to support trend analysis
- Analysed accident patterns by speed zone and local government area
- Built visualisations showing crash frequency and severity trends 
  using Matplotlib and Seaborn

## Key Findings
- Identified peak crash hours and high-risk days of the week
- Surfaced high-risk speed zones and local government areas
- Visualised severity trends to support transport safety decisions
